\section{Llama Data}
Figure~\ref{fig1} shows the following fit to the Llama data:
\begin{equation}
n_n(R) = \frac{n_n}{1+(R-R_{\rm sep})^{\,2}/L^{\,2}},
\end{equation}
where $n_n = 1.4\times 10^{17}\,{\rm m}^{-3}$, $L= 0.015\,{\rm m}$, $R_{\rm sep}= 1.92\,{\rm m}$.
