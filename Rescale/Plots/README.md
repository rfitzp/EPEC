# Asymptote scripts (see http://asymptote.sourceforge.net)

## filename.asy
- Usage:	    asy filename
- Produces:   filename.eps
---

	psi.asy  ... plots psi(R,Z)
	psi1.asy ... plots psi(R,Z) in which O-point and X-point positions recalculated
	t.asy    ... plots T(psi)
	ttp.asy  ... plots T T'(psi)
	p.asy    ... plots p(psi)
	pp.asy   ... plots p'(psi)
	q.asy    ... plots q(psi)
