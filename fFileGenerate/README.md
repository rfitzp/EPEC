# FFILEGENERATE

## Description

   Program to generate series of interpolated fFiles by calling
   the FLUX, NEOCLASSICAL, and PHASE codes.
	
## Requirements

   - blitz++ library (https://github.com/blitzpp/blitz)
   - gsl library (https://www.gnu.org/software/gsl)
   - BLAS library (http://www.netlib.org/blas)
   - asymptote (for plots) (https://asymptote.sourceforge.io)
   
## Contents

### /Source

- Makefile: GNU makefile
- *.f90: Fortran_90 source files
- *.cpp: C++ source files
	 
### /Inputs

- fFile.nml: Fortran_90 namelist file
